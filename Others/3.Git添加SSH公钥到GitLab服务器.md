# Git添加SSH公钥到git服务器

## 1. 首先在命令行输入 `ssh-keygen -t rsa -C "xxxx@xxx.xxx"` 来生成公钥

## 2. 从命令行拷贝出生成的公钥 
1. windows下：  
    clip < ~/.ssh/id_rsa.pub
2. Mac下：  
    pbcopy < ~/.ssh/id_rsa.pub
3. GUN/Linux：  
    xclip -sel clip < ~/.ssh/id_rsa.pub
    
此时公钥就已经拷贝到剪切板中。

## 3. 打开GitLab > Settings(右上角) > SSH Keys(左边菜单栏) >  Add SSH Key

## 4. 粘帖剪切板中的公钥到key文本域中，点击 Add key保存。
